# 📜 Prompteur Web Interactif

Un prompteur en ligne (Teleprompter) simple, fluide et ultra-léger, développé en HTML, CSS et JavaScript pur (Vanilla JS). Cet outil est idéal pour les créateurs de contenu, vidéastes, podcasteurs ou conférenciers ayant besoin de lire un texte face caméra sans quitter leur écran des yeux.

---

## ✨ Fonctionnalités

* **Interface Épurée :** Une zone d'édition simple pour coller votre script, basculant vers un écran noir à contraste élevé pour une lisibilité maximale.
* **Contrôle Total au Clavier :** Pilotez l'intégralité du défilement sans toucher à la souris.
* **Vitesse Ajustable :** Augmentez ou diminuez la vitesse de défilement en temps réel.
* **Taille de Police Personnalisable :** Ajustez dynamiquement la taille du texte à votre convenance pour un confort visuel optimal.
* **Indicateur de Vitesse :** Un affichage éphémère (5 secondes) vous indique le pourcentage de vitesse actuel lors de vos modifications.

---

## ⌨️ Raccourcis Clavier (Mode Prompteur)

Une fois le prompteur lancé, utilisez les touches suivantes :

| Touche | Action |
| :--- | :--- |
| **⬆️ Flèche Haut** | Faire défiler le texte vers le haut |
| **⬇️ Flèche Bas** | Faire défiler le texte vers le bas |
| **➡️ Flèche Droite** | Augmenter la vitesse de défilement |
| **⬅️ Flèche Gauche** | Diminuer la vitesse de défilement |
| **➕ Touche Plus (+)** | Grossir le texte |
| **➖ Touche Moins (-)** | Réduire le texte |
